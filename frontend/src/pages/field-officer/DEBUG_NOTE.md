# หมายเหตุ: กำลังแก้ปัญหา Geoman  

**ปัญหาที่เจอ:**  
1. ครั้งแรก: User สามารถแก้ไขขอบเขตหมู่บ้านได้ ❌  
2. หลังแก้: ไม่สามารถวาดอะไรบนแผนที่ได้เลย ❌  

**กำลังทำ:**  
ลดความซับซ้อนของการป้องกันลงเหลือแค่ `layer.pm.disable()` บน village layers  

**เป้าหมาย:**  
- ✅ วาดได้ปกติบนแผนที่  
- ✅ แต่แก้ไขขอบเขตหมู่บ้านไม่ได้
